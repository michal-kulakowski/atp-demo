Fortinet ATP Demo Installation Instructions

  1. Log onto your FortiSIEM instance (super) as root
  2. Clone github repository to convenient location
    • git clone git@github.com:michal-kulakowski/atp-demo.git
  3. Enter ATP demo directory
    • cd atp-demo
  4. Set up file based discovery
    • ./setup.sh
    • wait for the system to reboot

  5. Define SNMP credentials. Use any community string, discovery is going to run from disk anyway.

  6. Define IP ranges as shown below.

  7. Run discovery for both IP addresses. Make sure you get results.

  8. Load the rule
    • view rules.xml
    • copy contents
    • click Analytics > Rules > Import
    • paste & import
  9. Open doc/use.html with your browser and execute demo scenario described
